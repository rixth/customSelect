customSelect

A simple jQuery UI widget to pretty up select elements. It's fully unit tested too, you can run the tests if you like.

It works by setting the values back in your original select, so you can call the standard jQuery val methods, bind to the change event, etc right on the native element. It does however, also fire its own events.

It even works for multi selects. 

$('#yourSelect').customSelect();
